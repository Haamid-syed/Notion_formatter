Connect your workspace using the standard Notion OAuth flow. You must selectively grant access to specific pages—the system does not request full workspace clearance. Only selected index pages and their children are visible.

2. NON-DESTRUCTIVE EDITING

When a formatting trigger is invoked, the AI engine analyzes the content payload. It renders a new native Notion layout and appends it. If raw code dumps exist in the page, the system automatically infers the language runtime, segregates the code, and assigns it to a code_block property with native Notion syntax highlighting configured.
