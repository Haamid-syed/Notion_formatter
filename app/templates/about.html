ABOUT
THE PROJECT

PURPOSE // INTENT

Notion Formatter was built with one goal: to separate the cognitive load of writing from the tedium of formatting. By allowing humans to dump raw streams of consciousness into Notion, we leverage AI models to act as the editor—restructuring, formatting, and presenting the ideas in native Notion blocks without altering the core assertions.

It’s not just an API call; it’s an automated editorial pipeline built on a Brutalist architectural philosophy. Function over decoration. Structure over style.

TECHNICAL // STACK

  • Python / Flask Runtime Core
  • Notion REST API Integration
  • Google Gemini 2.5 Flash Parsing Engine
  • Custom AST-to-Notion block compiler
  • HTML/CSS Vanilla Frontend with GSAP Motion
  • Idempotent block manipulation
